# 项目标题

## Project setup
```
npm install
```

### 开启本地服务
```
npm run dev
```

### 打包
```
npm run build
```

### 发布静态资源
```
gulp publish
```

### 上线模板页面

在cms的[功能菜单]->[文章模板]栏目中新建模板，模板id必须以`_nodejs`为结尾，将`dist/index.html`内容复制进去。然后在测试栏目内发布文章选择该模板即可。
